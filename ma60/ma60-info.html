~ ~ ~ Welcome to MA60: Sharing is Caring! ~ ~ ~

This is a 13P setup where Power Roles and Mafia have a respective limit to their combined uses of their abilities, making them have to share and distribute properly those shots!

Setup

+ 7 Citizens + X-Shot Cop + X-Shot Jailkeeper + X-Shot Vigilante - Y-Shot Framer - Y-Shot Doctor - Y-Shot Rolecop X and Y are variables that start as 4 and increase by 1 each time a PR or Mafia dies (X increases if a PR dies; Y increases if a Mafia dies). PRâ€™s and Mafia cannot exceed a total combined shots across the game of X and Y, respectively, during a night. If X or Y is to be exceeded, all the corresponding actions by the team that night will fail, except for factional kill (neither failed action will count for the total shots used). To learn more about the setup: https://wiki.mafiascum.net/index.php?title=Sharing_is_Caring.

Voting Mechanics

Plurality: Enabled Majority: Enabled NoLynch: Enabled D2+ Ties: First to reach if No Lynch is not involved (No Lynch takes priority). No Votes: RNG

Game Mechanics

Day: 48 Hours Night: 24 Hours Mafia has a 24/7 private chat and can multitask.

Activity Requirements

Prods: At least 25 posts each 24 hours. - These 25 posts must be in relation to the game. - 0 posts in 24h will result in 2 prods. - If you get 3 prods, you will be replaced. Please only sign up for this game if you are able to commit an adequate amount of time to it (1h+ each day).

Rules

Please make sure to have read the Game Rules section of our #📜rules channel, so that you do not accidentally break one of them, since it can lead to punishments, such as game bans or forceful replacement, which I would like to avoid if possible.
